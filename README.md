# 画像生成AI用 日本語プロンプト変換ツール 🪄

## はじめに
このツールは、画像生成AIで画像を作成する際、イラストの中に「日本語の文字」をそのまま綺麗に描き込んでもらうための補助ツールです。

ブラウザ上で好きな日本語を入力するだけで、AIの翻訳システムをすり抜けるための「改行＋Unicode（記号の羅列）」の形式にサクッと変換できます。

## 作成の背景と前提
通常、画像生成AIに日本語で指示を出すと、裏側で高精度な自動翻訳（Google Flowなど）が入り、英語に変換されてから画像が作られます。これは非常に便利な機能なのですが、「看板や吹き出しの文字を、どうしても日本語のまま出力したい」という時には、文字まで英語に翻訳されてしまうジレンマがありました。

このツールは、テキストを一時的にUnicodeという世界共通の文字コードに変換し、さらに前後に改行を挟むことで、翻訳エンジンに「これは翻訳しなくていいデータだ」と認識させ、日本語のまま画像生成モデルに届けることを目的としています。

## 検証状況と確度について
私自身の環境にて、この「Unicode変換＋改行」の手法を用いることで、日本語がそのまま出力されやすくなることを検証しております。

ただし、AIのシステムや翻訳エンジンは日々アップデートを繰り返して進化しています。そのため、環境や時期によって「必ず100%成功する」と断定はできない点をご了承ください。現時点での「意図した文字を出すための、ちょっとした工夫」として、遊び心を持って使っていただければ幸いです。

## 使い方
1. 公開されているツールのURL（※ここにURLを貼ります）にアクセスします。
2. 上の入力欄に、画像に書き込みたい日本語（例：うっそー！）を入力します。
3. 「暗号化する（変換）」ボタンを押します。
4. 下の出力欄に変換されたテキスト（前後の改行を含む）が表示されるので、「コピーする」ボタンを押します。
5. コピーしたテキストを、普段お使いの画像生成AIのプロンプト入力欄にそのまま貼り付けて実行してください。

## ご注意
* 本ツールはブラウザ上でのみ動作し、入力したテキストが外部のサーバーに送信されることはありません。安心してお使いください。
